## 边界说明

本模块从语法、语义模型、类型理论到编译工程实践，对 JavaScript 与 TypeScript 进行全方位对比分析。内容涵盖类型擦除、运行时差异、编译器 API 等语言核心议题，是理解 TS 作为 JS 超集的本质边界的关键模块。

**非本模块内容**：框架选型对比、工具链对比、运行时平台（Node/Deno/Bun）对比。
