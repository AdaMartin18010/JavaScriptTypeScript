# 78-metaprogramming

Metaprogramming techniques and utilities implemented in TypeScript.

## Topics

| Topic | File | Description |
|---|---|---|
| Decorators | `decorators.ts` | Class/method/property/parameter decorators with metadata |
| Reflection | `reflection.ts` | Type introspection, object reflection, proxy factory |
| Code Generation | `code-generation.ts` | Template engine and TypeScript code generator |
| Proxy Interceptor | `proxy-interceptor.ts` | AOP-style proxy handlers for methods and properties |
| AST Traverser | `ast-traverser.ts` | Simplified object tree traversal with Visitor pattern |
| DI Container | `di-container.ts` | Lightweight dependency injection container with 3 lifetimes |
| Meta Techniques | `meta-techniques.ts` | Advanced metaprogramming patterns and tricks |

## Running Tests

npx vitest run 78-metaprogramming
