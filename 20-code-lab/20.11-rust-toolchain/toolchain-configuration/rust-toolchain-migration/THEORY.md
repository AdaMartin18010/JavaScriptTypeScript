## 二、设计原理

### 2.1 为什么存在

Rust 提供了内存安全和并发安全的保证，同时性能接近 C++。将其引入 JS 生态可以显著提升计算密集型模块的可靠性和性能。

### 2.2 权衡分析

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| napi-rs | 类型安全绑定 | 额外学习成本 | Node.js 原生扩展 |
| WASM 编译 | 跨平台、沙箱安全 | 性能开销与限制 | 浏览器端 |

### 2.3 与相关技术的对比

与 Node-API (C++) 对比：Rust 内存安全但学习曲线更陡。
