# CATEGORY.md — 02-design-patterns

## 所属维度

**语言核心（Language Core）**

## 边界说明

本模块收录软件工程中经过验证的经典设计模式在 JavaScript/TypeScript 中的实现。设计模式是语言无关的，但本模块聚焦于利用 JS/TS 特有的动态特性、闭包、原型链、类型系统来表达这些模式，属于语言能力的应用层面。

**非本模块内容**：前端组件模式（如 React 高阶组件）、特定框架的架构模式。
