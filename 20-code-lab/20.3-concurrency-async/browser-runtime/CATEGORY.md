# 模块归属声明

本模块归属 **「框架（Frameworks）」** 维度，聚焦浏览器运行时机制与底层原理。理解这些原理是掌握前端框架设计与性能优化的基础。

## 包含内容

- 事件循环（Event Loop）与任务调度
- V8 引擎执行模型
- 内存管理与垃圾回收策略
- 浏览器渲染管线（Rendering Pipeline）
- DOM 与虚拟 DOM 对比模型
