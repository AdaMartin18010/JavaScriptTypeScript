## 模块定位

> **维度：语言核心（Language Core）**
>
> 本模块属于语言核心中的**执行与并发维度**。内容聚焦 ECMAScript 规范及 HTML 规范定义的并发原语——Promise、async/await、事件循环、Worker、Atomics——而非某个框架的并发抽象。

## 模块概述

JavaScript 事件循环、异步 IO、并行计算与锁机制。通过可运行示例深入理解单线程事件驱动模型、宏任务/微任务调度、SharedArrayBuffer 与 Atomics 的多线程同步机制，以及流式并发控制模式。

## 核心主题

- 事件循环深度解析
- Promise 与 async/await
- Worker Threads
- Atomics 与 SharedArrayBuffer
- 并发控制模式
