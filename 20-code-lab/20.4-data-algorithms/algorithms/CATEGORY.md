# CATEGORY.md — 05-algorithms

## 所属维度

**语言核心（Language Core）**

## 边界说明

本模块实现经典算法并分析其复杂度。算法是语言无关的，但本模块聚焦于利用 JavaScript/TypeScript 的语言特性（如高阶函数、生成器、尾递归优化提示、TypedArray）来表达和优化这些算法。

**非本模块内容**：机器学习算法、特定领域的业务算法。
