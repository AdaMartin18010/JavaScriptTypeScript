# 维度标注

- **所属维度**: 应用领域（Application Domain）
- **应用域**: 低代码平台 — 可视化设计器、Schema 驱动与代码生成
- **模块编号**: 97-lowcode-platform

## 边界说明

本模块聚焦低代码平台的核心引擎实现，包括：
- 组件库管理与页面设计器（含撤销重做）
- Schema 定义与类型安全
- React/Vue 代码生成
- 工作流引擎与表达式引擎

通用 UI 组件库（Ant Design / MUI）和构建工具不属于本模块范围。
