# 低代码平台 — 理论基础

## 1. 低代码定义

低代码平台通过可视化界面和预置组件减少手写代码量：

- **可视化设计器**: 拖拽组件构建 UI
- **模型驱动**: 数据模型自动生成 CRUD 界面
- **逻辑编排**: 可视化流程图替代代码逻辑

## 2. 平台架构

```
设计时（Designer）
  ├── 组件库
  ├── 属性面板
  ├── 数据绑定
  └── 事件编排
         ↓ 生成/解析
运行时（Runtime）
  ├── 渲染引擎
  ├── 状态管理
  └── 动作执行器
```

## 3. 与相邻模块的关系

- **51-ui-components**: 组件库设计
- **56-code-generation**: 代码生成技术
- **53-app-architecture**: 应用架构模式
