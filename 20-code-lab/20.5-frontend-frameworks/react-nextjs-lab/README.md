# React + Next.js App Router 实验室

## 概述

本实验室提供 Next.js App Router 的端到端实践环境，包含5个里程碑项目。

## 里程碑

### M1: 基础路由与布局

- 文件系统路由约定
- 嵌套布局实现
- loading.tsx / error.tsx
- Parallel Routes 实验

### M2: RSC 与数据获取

- Server Component 数据获取
- React Cache 使用
- 避免请求瀑布
- Data Cache 配置

### M3: Streaming + Suspense

- Suspense 边界策略
- 骨架屏设计
- 渐进式加载
- View Transitions

### M4: Server Actions + 表单

- Server Actions 设计
- 表单验证
- 乐观更新
- 错误处理

### M5: 认证 + 边缘部署

- NextAuth.js / Clerk 集成
- Middleware 认证
- Edge Runtime 部署
- 性能监控

## 快速开始

```bash
cd milestone-01-basic-routing
npm install
npm run dev
```
