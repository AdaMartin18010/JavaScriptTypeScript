# 服务器优先前端实验室

## 概述

本实验室提供 HTMX + Alpine.js 的完整实践环境。

## 里程碑

### HTMX + Express (M1-M3)

- **M1**: 基础 AJAX 替换 — todo 列表 CRUD
- **M2**: 表单与验证 — 实时验证 + 服务器校验
- **M3**: 实时更新 — SSE 推送 + OOB 更新

### Alpine.js (M4-M5)

- **M4**: 交互组件 — 下拉菜单、模态框、标签页
- **M5**: 与后端 API 集成 — 购物车、表单状态

## 快速开始

```bash
cd htmx-express
npm install
npm run dev
```
