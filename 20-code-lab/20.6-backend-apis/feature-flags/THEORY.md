# 特性开关 — 理论基础

## 1. 特性开关定义

特性开关（Feature Flag）是一种允许在运行时控制功能可见性的技术：

- **无需部署即可发布/回滚功能**
- **支持渐进式发布和 A/B 测试**

## 2. 开关类型

| 类型 | 用途 | 生命周期 |
|------|------|---------|
| **发布开关** | 新功能灰度发布 | 功能稳定后移除 |
| **实验开关** | A/B 测试 | 实验结束后移除 |
| **运维开关** | 紧急关闭功能 | 长期保留 |
| **权限开关** | 按用户角色启用 | 长期保留 |

## 3. 发布策略

- **金丝雀**: 1% → 5% → 20% → 100%
- **按属性**: 地域、设备、用户群
- **按时间**: 定时发布（如黑色星期五）
- **Kill Switch**: 紧急情况下立即关闭功能

## 4. 与相邻模块的关系

- **65-analytics**: A/B 测试的数据分析
- **22-deployment-devops**: 部署策略
