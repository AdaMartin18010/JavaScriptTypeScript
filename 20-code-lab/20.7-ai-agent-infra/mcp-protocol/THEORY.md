## 二、设计原理

### 2.1 为什么存在

LLM 需要与外部世界交互才能解决实际问题。MCP 协议标准化了模型与工具、数据源之间的连接方式，降低了集成成本。

### 2.2 权衡分析

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| MCP 标准 | 工具生态共享 | 适配成本 | 通用 LLM 应用 |
| 直接集成 | 简单直接 | 重复开发 | 单一应用 |

### 2.3 与相关技术的对比

与直接 API 调用对比：标准化降低重复开发，直接调用更灵活。
