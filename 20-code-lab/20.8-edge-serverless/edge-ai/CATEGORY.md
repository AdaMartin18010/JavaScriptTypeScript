# 维度标注

- **所属维度**: 应用领域（Application Domain）
- **应用域**: 边缘 AI — 浏览器端推理、模型量化、WebNN/WebGPU 加速
- **模块编号**: 82-edge-ai

## 边界说明

本模块聚焦在边缘环境（浏览器、边缘节点）中进行 AI 推理，包括：
- 设备能力检测与模型量化
- ONNX Runtime 与 TinyML
- WebNN / WebGPU 计算加速
- 联邦学习基础

云端模型训练和大规模分布式训练不属于本模块范围。
