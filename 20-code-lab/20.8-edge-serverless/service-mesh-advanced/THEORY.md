# 高级服务网格 — 理论基础

## 1. 服务网格定义

服务网格是处理服务间通信的专用基础设施层：

- **数据平面**: Sidecar 代理（Envoy），拦截所有网络流量
- **控制平面**: 配置管理（Istiod），分发路由和安全策略

## 2. Sidecar 模式

每个 Pod 注入一个代理容器：

```
[App Container] ←→ [Envoy Sidecar] ←→ 网络
```

应用无感知地获得：

- mTLS 加密
- 流量管理（路由、分流、超时、重试）
- 可观测性（指标、追踪、日志）

## 3. 流量管理

- **虚拟服务**: 定义路由规则
- **目标规则**: 定义负载均衡策略
- **故障注入**: 模拟延迟和错误，测试韧性
- **熔断**: 连续失败时快速失败
