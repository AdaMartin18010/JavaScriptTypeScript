## 与知识库关联

每个模块均配有 `THEORY.md` 理论文档，位于模块根目录。理论与实践形成闭环：

```
20-code-lab/20.X-topic/
├── THEORY.md          # 理论说明
├── examples/            # 代码示例
└── exercises/           # 练习题
```
