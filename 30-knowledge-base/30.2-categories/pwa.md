# PWA（渐进式 Web 应用）

> PWA 技术栈与最佳实践。

---

## 核心技术

| 技术 | 说明 |
|------|------|
| **Service Worker** | 离线缓存、后台同步 |
| **Web App Manifest** | 安装配置、主题色 |
| **Web Push** | 推送通知 |
| **Background Sync** | 离线操作延迟同步 |
| **File System Access** | 本地文件读写 |

## 工具

| 工具 | 说明 |
|------|------|
| **Workbox** | Google 官方 SW 库 |
| **Vite PWA** | Vite 插件，自动生成 SW |
| **PWA Builder** | 微软官方 PWA 打包工具 |
