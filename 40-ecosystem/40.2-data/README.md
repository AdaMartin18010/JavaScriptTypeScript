# 生态数据追踪（Ecosystem Data Tracking）

> **关联**：`../40.3-trends/ECOSYSTEM_TRENDS_2026.md` | `data/`

## 数据维度

| 维度 | 说明 | 更新频率 |
|------|------|---------|
| GitHub Stars | 核心项目的 Stars 趋势 | 月度 |
| npm 下载量 | 周下载量（百万级） | 周度 |
| 版本发布 | 主要项目的版本发布记录 | 实时 |
| 安全漏洞 | CVE / GHSA 披露 | 实时 |
| 性能基准 | TechEmpower / 自定义基准 | 季度 |

## 数据来源

- npm registry API
- GitHub REST API
- TechEmpower Framework Benchmarks
- Snyk / OSV 漏洞数据库
