# 编译器工作坊

本工作坊带你从零实现一个迷你 TypeScript 类型检查器，理解编译器核心原理。

## 学习路径

| 阶段 | 主题 | 目标 |
|------|------|------|
| M1 | 迷你解析器 | 实现基础 AST 解析 |
| M2 | 类型检查器基础 | 实现基础类型推导 |
| M3 | 泛型推断 | 理解泛型约束与推断 |
| M4 | 条件类型 | 实现条件类型计算 |
| M5 | 类型体操 | 完成高阶类型挑战 |

## 快速开始

```bash
cd milestone-01-mini-parser
npm install
npm test
```
