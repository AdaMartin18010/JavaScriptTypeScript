# Lit 设计系统 Starter

## 概述

基于 Lit 的跨框架设计系统，包含 10+ 组件 + Storybook + 文档。

## 组件清单

- [x] Button — 变体、尺寸、状态
- [x] Input — 标签、验证、提示
- [x] Card — 标题、内容、操作
- [x] Modal — 动画、焦点陷阱
- [x] Tabs — 动态内容
- [x] Select — 单选/多选
- [x] Checkbox / Radio
- [x] Tooltip
- [x] Spinner / Loading
- [x] Toast / Notification

## 跨框架集成

- React 包装器
- Vue 包装器
- Angular 包装器

## 快速开始

```bash
npm install
npm run storybook
```
