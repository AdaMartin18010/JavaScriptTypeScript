**技术栈**：

- React Native (New Architecture)
- Expo Router (文件系统路由)

- TypeScript Strict

**功能特性**：

- 跨平台导航（iOS/Android）
- 原生 UI 组件
- 离线数据持久化（SQLite）
- 推送通知
