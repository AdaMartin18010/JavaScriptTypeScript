## 2. 极限与余极限的语义解释

> 🚧 **骨架占位符**：极限 = 模型的"交集"，余极限 = 模型的"联合"。

## 3. 初始对象与终端对象

## 4. 拉回与推出

> 🚧 **骨架占位符**：拉回 = 共同可表达行为，推出 = 并集的最小精化。
