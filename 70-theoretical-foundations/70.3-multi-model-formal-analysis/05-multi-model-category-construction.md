## 1. 模型范畴 Models 的构造

### 1.1 定义

构造范畴 $\mathbf{Models}$：

- 对象 = {操作语义, 指称语义, 公理语义, 类型系统, 运行时模型, React 模型, Vue 模型, ...}
- 态射 = 精化关系 / 模拟关系 / 编译映射

---

## 2. 极限与余极限的语义解释

### 2.1 极限 = 模型的"交集"

拉回（Pullback）= 两个模型的共同可表达行为。

### 2.2 余极限 = 模型的"联合"

推出（Pushout）= 两个模型的并集的最小精化。

---

## 3. 初始对象与终端对象

### 3.1 始对象

最抽象的模型（如纯数学语义）。

### 3.2 终对象

最具体的模型（如机器码执行）。

---

## 4. 拉回与推出

### 4.1 拉回

拉回 = 模型的交集。

### 4.2 推出

推出 = 模型的联合。

---
