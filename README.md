## 🚀 快速开始

```bash
cd jsts-code-lab
pnpm install

# 运行所有 Demo 示例
pnpm demo

# 或运行指定模块
pnpm tsx run-demos.ts design-patterns
pnpm tsx run-demos.ts consensus-algorithms
pnpm tsx run-demos.ts quantum-computing
pnpm tsx run-demos.ts autonomous-systems

# 运行测试
pnpm test

# 运行基准测试
pnpm benchmark
```
