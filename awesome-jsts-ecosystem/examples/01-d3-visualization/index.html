D3.js TypeScript 可视化示例

现代、类型安全、交互丰富的数据可视化组件

💡 快速开始

运行 npm install && npm run build 编译代码，然后点击下方的按钮查看各个图表示例。

柱状图 (Bar Chart)

支持渐变填充、交错动画、悬停提示框、网格线显示。点击按钮更新数据查看动画效果。
点击"渲染柱状图"按钮加载...

折线图 (Line Chart)

支持平滑曲线、区域填充、数据点标记、鼠标追踪线。适用于时间序列数据展示。
点击"渲染折线图"按钮加载...

饼图 (Pie Chart)

经典饼图展示，支持扇形分离、百分比标签显示、图例交互。点击扇形可高亮选中。
点击"渲染饼图"按钮加载...

环形图 (Donut Chart)

环形图变体，中心显示总计数值。适合展示占比关系同时保留中心区域用于信息展示。
点击"渲染环形图"按钮加载...

力导向图 (Force-Directed Graph)

基于物理模拟的网络图，支持节点拖拽、画布缩放、连接高亮。展示节点间的关系网络。
点击"渲染力导向图"按钮加载...
