## 标签体系规范

### TS 支持

| 图标 | 含义 | 说明 |
|------|------|------|
| 🟢 | TS 原生 | 使用 TypeScript 编写，类型定义完整且准确 |
| 🟡 | 类型完善 | 有优秀的 DefinitelyTyped 类型定义 |
| 🔴 | 弱类型 | 类型支持较弱或缺失 |

### 活跃度

| 图标 | 含义 | GitHub Stars |
|------|------|--------------|
| ⭐⭐⭐ | 极高活跃 | > 30k stars 或快速增长 |
| ⭐⭐ | 高活跃 | 10k-30k stars |
| ⭐ | 中等活跃 | < 10k stars 但稳定使用 |

### 维护状态

| 图标 | 含义 | 说明 |
|------|------|------|
| 🟢 | 活跃维护 | 持续更新，社区活跃 |
| 🟡 | 维护模式 | 稳定但更新频率降低 |
| 🔴 | 归档/废弃 | 不再维护或有更好的替代方案 |
