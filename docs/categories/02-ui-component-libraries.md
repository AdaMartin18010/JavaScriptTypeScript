## 📈 选型建议

### 按场景推荐

| 场景 | 推荐方案 | 理由 |
|------|----------|------|
| **快速启动新项目** | shadcn/ui | 现代、可控、无依赖负担 |
| **企业级中后台** | Ant Design / Semi Design | 生态完善，国内支持好 |
| **国际化产品** | MUI / Chakra UI | 全球社区，英文文档完善 |
| **移动端 H5** | Vant | 轻量，国内最流行 |
| **小程序+App** | Taro + NutUI / uni-app + uni-ui | 跨端能力最强 |
| **完全定制外观** | Headless UI + Tailwind | 完全可控的样式 |
| **无障碍要求高** | Radix UI / React Aria | 企业级无障碍支持 |

### 技术趋势

1. **Copy-Paste 模式兴起**：shadcn/ui 引领的新模式，让开发者完全掌控代码
2. **Headless UI 增长**：组件逻辑与样式分离成为主流
3. **设计系统下沉**：大厂设计系统（AntD、Semi、TDesign）向社区开放
4. **跨端统一**：一套组件适配多端的需求持续增长
5. **TypeScript 成为标配**：主流库均已原生支持 TS
