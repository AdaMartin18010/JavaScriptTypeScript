## 模块定位

> **维度：语言核心（Language Core）**
>
> 本模块聚焦 ECMAScript 语言规范的版本演进，属于语言核心中的**历史与规范维度**。内容严格围绕 TC39 提案与 ECMA-262 规范展开，不涉及框架特性或工具链演进。

## 模块概述

从 ES3 到 ES2026 的语言特性演进轨迹与关键决策。通过可运行代码示例，深入理解每个版本引入特性的语义动机、使用场景与潜在陷阱，建立对语言演进的系统性认知。

## 核心主题

- ES2015 革命性变革
- ES2017 async/await
- ES2020 BigInt/Optional Chaining
- ES2024/2025 新特性
- ES2026 前瞻
