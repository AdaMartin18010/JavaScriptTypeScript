## 模块定位

> **维度：语言核心（Language Core）**
>
> 本模块属于语言核心中的**模式与范式维度**。设计模式本身是语言无关的，但此处聚焦于利用 JavaScript/TypeScript 特有的动态特性、闭包、原型链与类型系统来表达经典模式，是语言能力在架构层面的集中体现。

## 模块概述

软件工程中经过验证的 23 种经典模式及其实现代码。涵盖创建型、结构型、行为型模式，以及 JavaScript 特有的函数式模式和 TS 类型级模式。所有实现均基于语言原生特性，不依赖特定框架。

## 核心主题

- 创建型模式
- 结构型模式
- 行为型模式
- 函数式模式
- 现代 JS 模式变体
