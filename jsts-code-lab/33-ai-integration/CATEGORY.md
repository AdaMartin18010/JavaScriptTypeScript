## 边界说明

本模块聚焦 AI 技术在 JS/TS 应用中的落地实现，包括：
- LLM API 调用与 Prompt 工程
- Embedding 与向量检索
- RAG 架构与 Function Calling
- Streaming 处理与 AI SDK 模式

底层模型训练、GPU 集群和 AI 框架本身的对比选型不属于本模块范围（请参见 `docs/categories/28-ai-agent-infrastructure.md`）。
