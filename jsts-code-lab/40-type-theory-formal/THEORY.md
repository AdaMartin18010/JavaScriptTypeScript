# Type Theory Formal (40-type-theory-formal)

本模块提供编程语言类型理论的可运行代码骨架，涵盖 λ 演算、简单类型系统、Hindley-Milner 推断和结构子类型。

## 目录

- `00-lambda-calculus/` - λ 演算基础
- `01-type-inference/` - HM 类型推断
- `02-subtyping/` - 结构子类型

## 关联模块

- `00-language-core/` - TypeScript 类型系统实践
- `10-js-ts-comparison/` - JS/TS 对比与类型理论
- `41-formal-semantics/` - 形式化语义
