# 69-cli-framework

CLI framework components implemented in TypeScript.

## Topics

| Topic | File | Description |
|---|---|---|
| CLI Builder | `cli-builder.ts` | High-level CLI builder with commands and options |
| Command Parser | `command-parser.ts` | Commander-like argument and flag parser |
| Argument Validator | `argument-validator.ts` | Declarative argument validation engine |
| Help Generator | `help-generator.ts` | Formatted help text generator |
| Progress Bar | `progress-bar.ts` | Terminal progress bar renderer |
| Interactive Prompt | `interactive-prompt.ts` | Readline-based interactive prompts |
| Config Loader | `config-loader.ts` | JSON/YAML configuration file loader |

## Running Tests

```bash
npx vitest run 69-cli-framework
```
