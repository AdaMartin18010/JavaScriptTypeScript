# 维度标注

- **所属维度**: 应用领域（Application Domain）
- **应用域**: NLP 工程 — 分词、情感分析、命名实体识别与语义搜索
- **模块编号**: 85-nlp-engineering

## 边界说明

本模块聚焦自然语言处理在 JS/TS 中的应用，包括：
- BPE 分词与文本预处理
- 情感分析与文本分类
- 命名实体识别与语义搜索
- Word Embedding 与 TF-IDF

大规模语言模型训练和 Transformer 架构原理不属于本模块范围。
