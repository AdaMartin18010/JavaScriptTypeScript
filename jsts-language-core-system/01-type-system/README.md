# 01 类型系统（Type System）

> 从基础类型到类型级编程，覆盖 TypeScript 类型系统的完整光谱

---

## 学习路线

```
基础层（必须掌握）
├── 01-foundations —— 原始类型、数组、元组、枚举、特殊类型
├── 02-type-inference-annotations —— 类型推断与注解
├── 03-interfaces-vs-type-aliases —— 接口与类型别名
├── 04-unions-intersections —— 联合与交叉类型
├── 05-narrowing-guards —— 类型收窄与守卫
└── 06-generics-deep-dive —— 泛型

进阶层（高级开发）
├── 07-mapped-types —— 映射类型
├── 08-conditional-types —— 条件类型
├── 09-template-literal-types —— 模板字面量类型
├── 10-utility-types-patterns —— 工具类型与模式
└── 11-type-level-programming —— 类型级编程

专家层（架构/库开发）
├── 12-variance —— 协变/逆变/双变/不变
├── 13-structural-vs-nominal —— 结构化 vs 标称类型
├── 14-type-soundness-boundary —— 类型健全性边界
├── 15-ts5-ts6-new-type-features —— TS 5.x/6.0 新特性
└── 16-ts7-go-compiler-preview —— TS 7.0 Go 编译器前瞻
```
