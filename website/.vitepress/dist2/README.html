Awesome JS/TS Ecosystem 文档站点 ​

基于 VitePress 构建的文档站点。

本地开发 ​

bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建
npm run docs:build

# 预览构建结果
npm run docs:preview

目录结构 ​

website/

├── research/             # 研究报告
├── templates/            # 文档模板
├── index.md              # 首页
└── about.md              # 关于页面
