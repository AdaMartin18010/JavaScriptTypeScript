新增实验室专题 ​

除原有模块外，代码实验室还包含以下深度专题实验：

语言核心实验室 ​

  • 语言核心实验室 — 变量提升 vs TDZ、闭包内存泄漏检测、原型链遍历、this 绑定规则的交互实验
  • 异步并发实验室 — Promise 微任务队列、宏任务 vs 微任务、async/await 展开、并发控制
  • 类型系统实验室 — 条件类型推断、映射类型键重映射、模板字面量类型、Branded Types

形式化语义实验室 ​

  • Lambda 演算实验室 — 从 λ 演算基础到 JavaScript 箭头函数、高阶函数与闭包的对应关系
  • 操作语义实验室 — 小步语义 vs 大步语义、状态转换规则、环境模型可视化
  • 公理语义实验室 — Hoare 三元组、循环不变量、分离逻辑与类型守卫

运行时与系统实验室 ​

  • 变量系统实验室 — 词法作用域 vs 动态作用域、变量提升 TDZ、闭包捕获、垃圾回收机制
  • 子类型关系实验室 — 宽度子类型、深度子类型、函数子类型、递归类型与方差
  • 控制流实验室 — 顺序/选择/循环形式化、异常控制流、生成器/迭代器、async 生成器

工程与框架实验室 ​

  • 工程环境实验室 — Vite 配置优化、TypeScript 严格模式、ESM/边缘运行时
  • Server Functions 实验室 — Server Actions 形式化、RPC 边界类型安全、流式响应、边缘计算
  • Mini TS 编译器实验室 — 词法分析、语法分析、类型检查、代码生成的完整流水线
  • 认证授权实验室 — JWT/OAuth2 流程、RBAC/ABAC、会话管理、MFA 与 Passkeys

实验方法论 ​

每个实验室遵循统一的方法论框架：

阶段目标输出
理论背景建立概念直觉形式化定义 + 直观类比
实验代码可运行的最小示例完整代码 + 环境要求
预期结果验证理论预测输出截图 + 解释
变体探索边界条件与反例修改后的代码 + 不同结果
总结归纳形成可迁移的知识速查表 + 最佳实践
