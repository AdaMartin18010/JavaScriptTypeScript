Stars 趋势报告 ​

生成时间: 2026/5/2 10:25:02 | 追踪仓库数: 53 | 分类数: 9

方法论说明 ​

数据来源 ​

  • GitHub REST API repos/{owner}/{repo} 端点
  • 数据采集频率：每月初
  • Stars 数据为公开仓库的 stargazers_count

增长率计算 ​

月增长率 = (本月Stars - 上月Stars) / 上月Stars × 100%
年增长率 = (本月Stars - 去年同期Stars) / 去年同期Stars × 100%
1
2

局限性 ​

局限说明
Stars ≠ 采用率Stars 反映兴趣而非生产使用
时间偏差新项目基数小，增长率天然更高
地域偏差GitHub 用户以欧美为主
企业项目企业内部项目不在统计范围内
