# 关于本站

## 项目简介

**Awesome JS/TS Ecosystem** 是一个精心策划的 JavaScript/TypeScript 生态系统资源列表，旨在帮助开发者：

- 🎯 **快速了解** 主流框架和工具的特点与适用场景
- 📊 **对比选择** 不同技术方案的优缺点
- 🚀 **跟踪趋势** 掌握前端生态的最新动态
- 💡 **最佳实践** 学习业界推荐的开发模式

## 收录标准

本项目收录的资源需满足以下条件：

| 标准 | 说明 |
|------|------|
| ⭐ Stars | GitHub Stars > 1000（特殊优秀项目除外） |
| 🔄 维护状态 | 最近 6 个月内有更新 |
| 📘 TypeScript | 原生支持或提供类型定义 |
| ✅ 生产就绪 | 有实际生产环境使用案例 |

## 分类体系

### 📦 P0 - 前端核心
1. 🖥️ **Frontend Frameworks** - React, Vue, Angular, Svelte, Solid
2. 🎨 **UI Component Libraries** - shadcn/ui, Ant Design, MUI, Chakra
3. ⚡ **Build Tools** - Vite, Webpack, Rollup, esbuild
4. 📊 **Data Visualization** - D3.js, ECharts, Chart.js, Three.js
5. 🗂️ **State Management** - Zustand, Redux, Jotai, Pinia
6. 🛣️ **Routing** - React Router, TanStack Router, Vue Router

### 🛠️ P1 - 工程化
7. 🎭 **SSR/Meta Frameworks** - Next.js, Nuxt, Remix, Astro
8. 📝 **Form Handling** - React Hook Form, Formik
9. ✅ **Validation** - Zod, Yup, Joi
10. 🎨 **Styling** - Tailwind CSS, Styled Components

### 🔧 P2 - 后端与工具
11. 🗄️ **ORM/Database** - Prisma, Drizzle, TypeORM
12. 🧪 **Testing** - Vitest, Jest, Playwright
13. 🛠️ **Linting** - ESLint, Prettier, Biome
14. 🔐 **Backend** - Express, Fastify, NestJS
