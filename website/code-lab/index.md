## 新增实验室专题

除原有模块外，代码实验室还包含以下深度专题实验：

- [类型系统实验室](./type-system-lab.md) — 条件类型推断、映射类型键重映射、模板字面量类型、Branded Types

### 形式化语义实验室

- [Lambda 演算实验室](./lab-00-lambda-calculus.md) — 从 λ 演算基础到 JavaScript 箭头函数、高阶函数与闭包的对应关系

### 运行时与系统实验室

- [变量系统实验室](./lab-02-variables.md) — 词法作用域 vs 动态作用域、变量提升 TDZ、闭包捕获、垃圾回收机制
- [子类型关系实验室](./lab-02-subtyping.md) — 宽度子类型、深度子类型、函数子类型、递归类型与方差
- [控制流实验室](./lab-03-control-flow.md) — 顺序/选择/循环形式化、异常控制流、生成器/迭代器、async 生成器

### 工程与框架实验室

- [工程环境实验室](./lab-01-basic-setup.md) — Vite 配置优化、TypeScript 严格模式、ESM/边缘运行时
- [Server Functions 实验室](./lab-02-server-functions.md) — Server Actions 形式化、RPC 边界类型安全、流式响应、边缘计算
- [Mini TS 编译器实验室](./lab-03-mini-typescript.md) — 词法分析、语法分析、类型检查、代码生成的完整流水线
- [认证授权实验室](./lab-03-auth.md) — JWT/OAuth2 流程、RBAC/ABAC、会话管理、MFA 与 Passkeys

## 实验方法论

每个实验室遵循统一的方法论框架：

```mermaid
flowchart LR
    A[理论背景] --> B[实验代码]
    B --> C[预期结果]
    C --> D[变体探索]
    D --> E[总结归纳]
    style A fill:#4ade80,color:#000
    style E fill:#f472b6,color:#000
```

| 阶段 | 目标 | 输出 |
|------|------|------|
| **理论背景** | 建立概念直觉 | 形式化定义 + 直观类比 |
| **实验代码** | 可运行的最小示例 | 完整代码 + 环境要求 |
| **预期结果** | 验证理论预测 | 输出截图 + 解释 |
| **变体探索** | 边界条件与反例 | 修改后的代码 + 不同结果 |
| **总结归纳** | 形成可迁移的知识 | 速查表 + 最佳实践 |

## 环境准备

### 本地运行

```bash
# 克隆项目
git clone https://github.com/AdaMartin18010/JavaScriptTypeScript.git
cd JavaScriptTypeScript

# 安装依赖
npm install

# 启动开发服务器
cd website
npm run docs:dev
```
