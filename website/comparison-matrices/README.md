1. 先做前端/后端框架选型，确定技术栈主基调。
2. 结合 SSR 元框架与构建工具对比，完善工程化链路。
3. 参考状态管理与 UI 库对比，细化应用层基础设施。
4. 最后阅读测试工具与 ORM 对比，补齐质量与数据层方案。
