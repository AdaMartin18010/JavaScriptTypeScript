## 技能要求趋势

### 必备技能

- React 18+（Hooks、Concurrent Features）
- TypeScript（严格模式）
- Next.js App Router
- Tailwind CSS / CSS-in-JS

### 加分技能

- React Server Components
- Streaming / Suspense
- Edge Runtime
- Vercel AI SDK
- React Compiler
