## 一、分析框架

### 1.1 维度定义

| 维度 | 说明 | 权重 |
|:---|:---|:---:|
| **技术准确性** | 源码引用、API 签名、算法步骤是否正确 | 🔴 高 |
| **时效性** | 版本对齐、发布日期、未来路线图 | 🔴 高 |
| **覆盖度** | 概念定义、边界情况、反例、异常路径 | 🟠 中 |
| **国际对齐** | 与英文权威信源的一致性、术语翻译 | 🟠 中 |
| **思维表征** | 图表、决策树、矩阵、推理路径可视化 | 🟡 中 |
| **可验证性** | 形式证明、实验数据、可复现基准 | 🟡 中 |

### 1.2 对称差分类

```
对称差 = 现有独有内容 ⊕ 网络最新独有内容

现有独有 (A \ B):
  → 可能是"过时的"或"本地化的"
  → 处理策略: 验证后保留、更新或删除

网络最新独有 (B \ A):
  → 缺失的内容
  → 处理策略: 补充、整合、本地化
```
