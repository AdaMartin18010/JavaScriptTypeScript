# TypeScript 类型系统深度掌握 — 练习题

## 目录结构

```
exercises/
├── easy/       # 20 题 — 基础泛型、条件类型、类型推断
├── medium/     # 20 题 — 映射类型、模板字面量、类型守卫
├── hard/       # 15 题 — 递归类型、类型体操、真实场景
└── extreme/    # 5 题  — 极限挑战（JSON Parser、类型算术）
```

## 使用方式

1. 按难度逐级完成
2. 每题限时 15-30 分钟
3. 参考 `../07-type-challenges-workbook.md` 的解题模式
4. 完成后对照官方解答
